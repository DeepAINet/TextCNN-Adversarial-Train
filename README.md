# TextCNN-Adversarial-Train
PGD, Free, FGSM used in TextCNN


# 中文数据集
从[THUCNews](http://thuctc.thunlp.org/)中抽取了20万条新闻标题,抽样了部分数据集。

类别：财经、房产、股票、教育、科技、社会、时政、体育、游戏、娱乐。

# 数据集划分：
|数据集|数据量|
|-----|-----|
|训练集  | 54000 |
|验证集  |  3000 |
|测试集  |  3000  |
